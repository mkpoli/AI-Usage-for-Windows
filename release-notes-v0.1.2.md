## AI Usage v0.1.2

Antigravity and Gemini model quota update.

### Highlights

- Antigravity now preserves each visible model quota bucket instead of merging models into broad Gemini or Claude groups.
- Model-specific lines can now show Gemini 3.5 Flash, Gemini 3.1 Pro, Claude Sonnet 4.6, Claude Opus 4.6, and GPT-OSS quota entries separately when Antigravity exposes them.
- Gemini now preserves model-specific Gemini 3.x quota bucket labels when Google exposes them through Gemini CLI / Code Assist quota APIs.
- Documentation clarifies that Gemini app compute-based limits and Gemini API token accounting are separate from the Gemini CLI / Code Assist quota data shown in AI Usage.
- Home Dashboard now includes Gemini 3.1 preview quota labels and Antigravity's focused Gemini model quota labels.
- Antigravity duplicate model labels are collapsed to one line.
- App and installer versions updated to 0.1.2.
